# AgentLLM Demo Agents

Demo agents showcasing AgentLLM features and best practices.

## Agents

### DemoAgent

Simple agent demonstrating:
- Required toolkit configuration (favorite color)
- Utility tools (color palette generation)
- Session memory
- Extensive logging

## Installation

```bash
pip install agentllm-agents-demo
```

## Usage

The agent is automatically discovered when installed.

See [agentllm-core documentation](https://github.com/yourorg/agentllm-core) for usage details.
